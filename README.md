# Análisis de los pasajeros del Titanic

Este proyecto tiene como objetivo estudiar y analizar los datos de los pasajeros del Titanic.

A lo largo del trabajo se explorarán distintas características de los pasajeros, como por ejemplo:
- Edad
- Sexo
- Clase del billete
- Puerto de embarque
- Supervivencia

El análisis se realizará utilizando Python y sus librerías habituales para el tratamiento y análisis de datos.

Este repositorio se encuentra en una fase inicial y se irá ampliando progresivamente con nuevos análisis y conclusiones.
